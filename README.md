# Geneticure.com

## Folders of note

-   `/public/img`: All the images used on the site.
-   `/src/pages`: Each file represents one of the routes within geneticure.com, e.g. `rdn.astro` is `geneticure.com/rdn`. Note that `index.astro` is `geneticure.com/`.
-   `/src/sections`: Parts of the above pages.
-   `/src/text`: Things that are displayed in lists (e.g. citations, FAQs, patient quotes, etc.), and long-form text (the Covid and RDN articles, Terms of Service, etc.).
